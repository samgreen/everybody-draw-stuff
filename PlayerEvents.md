Player Events
====================

* `updateColor`
  * `color` - String
* `startPaint`
  * `position` - Dictionary
    * `x` - x position
    * `y` - y position
* `movePaint`
  * `position` - Dictionary
    * `x` - x position
    * `y` - y position
* `stopPaint`
  * `position` - Dictionary
    * `x` - x position
    * `y` - y position
* `setName` - String to set the player's name
* `busy` - Player is on a menu
